0. 새로운 폴더 하나 만들기
1. 그 폴더 안에서 git clone https://github.com/hyunj24/my_team_blog.git 명령어 실행
2. vue create my_team_blog // git clone받은 path에 설치
    1) 설치 시 merge선택
    2) vue2랑 vue3 중 vue3 선택 (가장 최신 버전)
3. npm run serve 실행 후 브라우저 localhost:8080 로 블로그 페이지 확인
(4, 5번은 Docker로 실행하는 법, 근데 5분 정도 걸릴 수 있음)
4. docker 엔진 켜고 docker-compose build 실행 
5. docker-compose up -d 실행 후 브라우저 localhost:8080 로 블로그 페이지 확인
